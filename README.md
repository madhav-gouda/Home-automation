# 🏠 AVENIR AUTOMATION

### Smart Home Automation Web Platform

---

## 📌 Overview

**AVENIR AUTOMATION** is a smart home automation web application designed to improve modern living through intelligent solutions.
It enhances **comfort, security, and energy efficiency** using automation technologies.

### ✨ Features

* Smart Lighting Control
* Advanced Security Systems
* Voice-Controlled Automation
* Remote Access via Web
* Energy Optimization

---

## 🛠️ Technologies Used

**Frontend:**

* HTML5
* CSS3
* JavaScript

**Backend:**

* Python (Flask)

---

## 🖥️ Project Structure

```
avenir-automation/
│
├── templates/        # HTML files  
├── static/           # CSS, JS, images

```

---

## ⚙️ How to Run

```bash
git clone https://github.com/yourusername/avenir-automation.git
cd avenir-automation
pip install flask
python app.py
```

Open in browser:
http://127.0.0.1:5000
